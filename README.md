# 🧠 AI Workflow Builder

> Instantly turn your commands or sheets into beautiful, shareable visual workflows — in just one click.

AI Workflow Builder is a powerful tool that converts your natural language instructions or structured sheets into diagrammatic workflows. Built for creators, developers, and teams who want to visualize processes instantly and share them across platforms like Notion, Email, Twitter, and more.

---

## 🚀 Features

- One-click conversion from command or sheet to diagram  
- Accepts natural language or spreadsheet-like data  
- Clean, drag-and-drop user interface  
- Share workflows via Notion, Email, or social media  
- Export diagrams as images (coming soon)  
- Powered by React Flow and modern AI APIs  

---

## 🛠️ Tech Stack

### Frontend
- React  
- JavaScript  
- Tailwind CSS  
- React Flow   

---

## 📦 Installation

```bash
git clone https://github.com/Anubhav505/FLOWGEN.git
cd ai-workflow-builder
npm install
```
## ▶️ To start the development server

```bash
npm run dev
```
